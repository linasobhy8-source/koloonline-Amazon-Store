**Koloonline Amazon Store** هو متجر أفلييت ديناميكي يعرض منتجات Amazon بطريقة احترافية مع نظام تتبع بسيط باستخدام LocalStorage وتجربة مستخدم قوية، مصمم لزيادة التحويلات والمبيعات.

## 🚀 المميزات

- 🔎 نظام عرض منتجات احترافي
- 🧠 صفحات بيع محسنة للتحويل (Sales Pages)
- 🛒 سلة مشتريات ديناميكية
- 🎉 صفحة شكر بعد الشراء (Thank You Page)
- 📊 تتبع بسيط للطلبات باستخدام LocalStorage
- ⚡ أداء سريع بدون سيرفر (Frontend Only)
- 📱 تصميم متجاوب لكل الأجهزة
- 🔍 SEO جاهز للفهرسة في Google
- 💰 قابل للتطوير لاحقًا إلى SaaS

## 💡 فكرة المشروع

المشروع يهدف إلى:

1. عرض منتجات Amazon Affiliate
2. زيادة النقرات والتحويلات
3. تحسين تجربة المستخدم لرفع الأرباح
4. بناء متجر بسيط قابل للتطوير لاحقًا إلى SaaS

## 🧰 التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage للتخزين
- GitHub + Vercel للنشر
