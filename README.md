## 🚀 التقنية المستخدمة

- Next.js (SSR + SEO Friendly)
- Firebase Firestore (Products Database)
- Vercel Hosting
- Google Analytics 4 (GA4)
- Google Tag Manager (GTM)
- Amazon Affiliate System
